Environment API Endpoints


The system exposes the following endpoints for programmatic control:


Method Endpoint Description
POST /reset Resets the simulation to the initial state.
POST /step Executes an action and advances the environment.
POST /run_demo Run a full autonomous episode and return the trace.
GET /health Checks the status of the environment server.
